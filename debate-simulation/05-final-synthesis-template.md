## 风险与缓解

| 风险 | 严重度 | 概率 | 缓解措施 |
|------|--------|------|---------|
| 平台 UI 变化导致选择器失效 | 🔴 高 | 🟡 中 | 选择器 YAML 外部化 + 多级 fallback + 健康检查 API |
| 平台反爬检测导致封号 | 🔴 高 | 🟢 低 | 使用持久化 profile 模拟正常用户 + 合理间隔 |
| 单平台故障导致辩论中断 | 🟡 中 | 🟡 中 | 优雅降级，最少 2 AI 可继续 |
| AI 响应超时 | 🟡 中 | 🟡 中 | 120s 超时 + 重试 3 次 + 指数退避 |
| 登录 Cookie 过期 | 🟡 中 | 🟡 中 | 辩论前自动检测 + 提示用户重新登录 |
